Geolocation

This project was made in my free time. My father cooperates with the archive of Geel, and he wanted a way to show where a plot used to be located. This project is just in its baby shoes, because there are 6 other maps left to reference.

Working at this project I have learned how to reference an old map with a more up‐to‐date map or vice versa. I again learned how to interact with a client, even if it was my father (and indirect to the archive) we had to communicate and make decisions. This project was made in QGIS which has a lot to offer. With this programme you can reference, draw shape layers, select data and a possibility to export it. Down below, you can find a full demo of the project.
