Context

This project was made in my free time. My father cooperates with the archive of Geel and he wanted a way to show where a plot used to be located. This project is just in it's baby shoes, because there are 6 other maps left to reference.

When you click on a plot (green), house (red), road (brown), river or lake (blue) the programme will show corresponding data like: the name, size, owner and type. For now this project is only loacted in 1834, but when this is done more versions will be added.

Conclusion

Working at this project I have learned how to reference an old map with a more up‐to‐date map or vice versa. I again learned how to interact with a client, even if it was my father (and indirect to the archive) we had to communicate and make decisions.

This project was made in QGIS which has alot to offer. With this programme you can refercence, draw shapelayers, select data and a possibility to export it. Down below you can find a full demo of the project.
